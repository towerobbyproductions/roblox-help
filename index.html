🔗 Roblox Universe API

Our Universal ID Finder uses the official Roblox API endpoint:

https://apis.roblox.com/universes/v1/places/{placeId}/universe

This powerful API endpoint allows developers to retrieve the Universe ID for any Roblox game using its Place ID. The Universe ID is essential for:

  • Accessing game data through Roblox APIs
  • Analytics and game statistics
  • Cross-game development
  • Third-party integrations
